# Quality Care Dental — Landing Page

A high-fidelity, production-quality landing page for **Quality Care Dental**, a premium family dental practice in Auckland, New Zealand. This lead-generation website is built with a cinematic aesthetic, smooth animations, and optimized for high-performance rendering.

## 🌟 Key Features

- **Cinematic Experience**: UHD video backgrounds and high-definition imagery.
- **Butter-Smooth Interaction**: Integrated **GSAP** and **Lenis** for refined, momentum-based scrolling.
- **Responsive Mastery**: Tailored experiences for Desktop, Tablet, and Mobile.
- **Lead Focused**: Optimized consultation booking system and clear conversion paths.
- **Modern Stack**: HTML5, Vanilla CSS, Tailwind CSS v3, GSAP, and ScrollTrigger.

## 🛠️ Technology Stack

- **Styling**: Tailwind CSS, Custom Vanilla CSS.
- **Animations**: GSAP (GreenSock Animation Platform), ScrollTrigger.
- **Scrolling**: Lenis Smooth Scroll.
- **Design System**: Editorial typography (Cormorant Garamond & Jost) and a sophisticated natural palette.

## 🚀 Deployment

This project is optimized for deployment on **Vercel**.

1. Connect this repository to your Vercel account.
2. The project uses standard HTML/CSS/JS; no special build steps are required.
3. Production headers and caching for video assets are pre-configured in `vercel.json`.

## 📂 Project Structure

- `index.html`: Core semantic structure.
- `style.css`: Design system, animations, and responsive rules.
- `main.js`: Interactive logic and animation choreography.
- `*.mp4`: High-definition cinematic video backgrounds.
